# Major Lessons Learned

## Git vs GitHub

Git is the version control system.

GitHub is a hosting platform.

---

## Modern Python Projects

Modern Python projects use:

- pyproject.toml
- src layout
- virtual environments

rather than relying on older project structures.

---

## Virtual Environments

A virtual environment is not another Python installation.

Instead, activation temporarily changes PATH so Python and pip resolve to the project-specific executables.
