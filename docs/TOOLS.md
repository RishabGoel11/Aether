# Tools Used in Aether

## Ruff

Purpose:
Linting and formatting.

Chosen Because:
- Extremely fast
- Modern Python standard
- Replaces multiple older tools

Alternatives:
- Black
- Flake8
- isort

## Pytest

Purpose:
Automated testing.

Chosen Because:
- Industry standard
- Simple syntax
- Excellent plugin ecosystem

Alternatives:
- unittest
- nose2
